# 📦 Quick Ecommerce — Multi-Vendor, Multi-Store Next.js eCommerce Platform

A fast, modern, SEO-optimized eCommerce frontend built with **Next.js + TypeScript + Tailwind CSS**.
Designed for multi-vendor and multi-store marketplaces — groceries, electronics, fashion, furniture, and more.

## 🚀 Features

### 🛒 Ecommerce

* Multi-vendor + multi-store support
* Product listings, variations & filters
* Cart, checkout & order flow
* Categories & collections
* Search with autocomplete

### 👥 User Roles

* Customer
* Vendor / Seller
* Admin
* Delivery (optional)

### ⚙️ System

* Next.js App Router (SSR & SEO-friendly)
* Fully responsive UI
* Tailwind CSS modern styling
* TypeScript for reliability
* Modular and easy to customize
* Ready for integrations (API, payments, auth)


## 📁 Folder Structure

```
/
├─ src/              # App Router pages, components, hooks
├─ public/           # Static assets
├─ types/            # TypeScript definitions
├─ scripts/          # Utility scripts
├─ next.config.mjs
├─ tailwind.config.ts
├─ tsconfig.json
├─ server.js         # Optional custom Node server
├─ .env.sample
└─ package.json
```

## 🔧 Installation

1. Install dependencies

   ```bash
   npm install
   ```

2. Create your environment file

   ```bash
   cp .env.sample .env
   ```

   Update API URLs and keys based on your backend.

3. Run the project in development

   ```bash
   npm run dev
   ```


## 🚀 Build for Production

```bash
npm run build
npm run start
```

(Optional custom server)

```bash
node server.js
```


## 🧩 Customization

You can customize everything easily:

* Components → `src/components`
* Styles → `src/styles` + Tailwind config
* API logic → `src/lib`
* Routes → `src/app`
